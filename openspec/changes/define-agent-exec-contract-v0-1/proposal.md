# 変更提案: agent-exec v0.1 契約仕様の固定

## 概要

agent-exec v0.1 の CLI 契約（サブコマンド、非対話、stdout JSON-only、共通レスポンス、終了コード、ヘルプ言語）を固定する。

## 目的

- エージェント/自動化が安定して呼び出せる外部契約を確定する
- JSON 出力の安定性を担保し、stdout/stderr の分離を保証する

## スコープ

- CLI サブコマンド構成（run/status/tail/wait/kill）
- JSON 共通エンベロープとエラー形式
- 非対話・stdout JSON-only のルール
- 終了コードの規約
- ヘルプ表記の英語固定

## スコープ外

- 実行監視の内部構成（監視プロセスなど）
- ジョブ保存先やログ形式の詳細
- Windows のプロセスツリー実装

## 分割方針

この変更は CLI の外部契約に限定し、実行管理・ジョブ保存・Windows 実装・テスト/CI とは分離する。
