MCP and HTTP

MCP stdio

agent-exec mcp exposes managed-job tools: run, status, tail, wait, and kill. A disconnected client does not stop the managed job; reconnect with its job_id.

AGENT_EXEC="$(command -v agent-exec)"
"$AGENT_EXEC" mcp

HTTP

Use agent-exec serve to expose the same managed-job operations over HTTP. Keep the server on a trusted network and use its documented authentication and CORS settings.

agent-exec serve --help
