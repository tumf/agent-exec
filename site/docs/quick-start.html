Quick start

Start a job, retain its stable job_id, then wait for a bounded completion result.

agent-exec run --until 1 -- sh -c 'echo started; sleep 3; echo finished'
# Copy the job_id from the JSON response:
agent-exec wait <job_id> --until 5

The commands use only agent-exec and a POSIX shell; no JSON parser is required. A successful wait returns a terminal state, exit_code: 0, and bounded stdout and stderr. Run agent-exec tail <job_id> for later or repeated log retrieval.
