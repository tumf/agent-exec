Quick start

Start a job, retain its stable job_id, then wait for a bounded completion result.

START_FILE=$(mktemp)
agent-exec run --until 1 -- sh -c 'echo started; sleep 3; echo finished' >"$START_FILE"
JOB_ID=$(python3 -c 'import json, sys; print(json.load(open(sys.argv[1]))["job_id"])' "$START_FILE")
agent-exec wait "$JOB_ID" --until 5

A successful wait returns a terminal state, exit_code: 0, and bounded stdout and stderr. Run agent-exec tail "$JOB_ID" for later or repeated log retrieval.
