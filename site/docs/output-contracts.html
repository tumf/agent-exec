Output contracts

Response-producing CLI commands write exactly one JSON object to stdout by default. Pass --yaml for YAML. Diagnostic logs go to stderr and are controlled by RUST_LOG, -v, and -vv.

Responses include a schema version, success flag, response type, and job fields appropriate to the command. Output excerpts are bounded; log paths identify complete retained logs.

Generated completions, MCP stdio protocol, the HTTP server, startup configuration errors, and Clap help/version output are not response envelopes.
