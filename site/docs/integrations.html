Agent integrations

Resolve an absolute executable path because MCP clients may not inherit an interactive shell PATH.

AGENT_EXEC="$(command -v agent-exec)"
test -x "$AGENT_EXEC"

Claude Code

claude mcp add --scope local agent-exec -- "$AGENT_EXEC" mcp

Codex CLI

codex mcp add agent-exec -- "$AGENT_EXEC" mcp

OpenCode

{
  "mcp": {"agent-exec": {"type":"local", "command":["/absolute/path/to/agent-exec", "mcp"], "enabled":true}}
}

Hermes Agent

hermes mcp add agent-exec --command "$AGENT_EXEC" --args mcp

Without MCP, install the embedded CLI skill with agent-exec install-skills.
